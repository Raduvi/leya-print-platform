Leya Print Server

There's nothing at the root of a leya print server

you might be looking for:

  • /pdf/tpl-name/filename.pdf - the pdf backend service
  • /pdf/alive - pdf service alive/health check
  • /tpl/alive - template service alive/health check
  • /auth/alive - authentication service alive/health check
  • /tpl/ - template backend service
  • /dev/ - the developer interface
